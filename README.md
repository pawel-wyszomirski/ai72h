## Instalacja

1. Sklonuj repozytorium
2. Zainstaluj zależności: `pip install -r requirements.txt`
3. Utwórz plik `.env` i dodaj wymagane klucze API:
OPENAI_API_KEY=your-key
TAVILY_API_KEY=your-key
